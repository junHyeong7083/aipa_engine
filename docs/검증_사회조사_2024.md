# AIPA 검증 - 통계청 사회조사 2024 비교

> AIPA가 시뮬레이션한 응답 분포를 통계청 사회조사 2024년 실제 결과(13세 이상 36,000명)와 비교하여 정확도 검증

---

## 표준 문항 (통계청 공식 wording)

> **"귀하는 가족 관계에서 다음 각 항목에 대하여 어느 정도 만족하고 계십니까?"**

**5점 척도 보기:**
1. 매우 만족
2. 약간 만족
3. 보통
4. 약간 불만족
5. 매우 불만족

**조사 영역 5개:**
- 전반적인 가족 관계
- 배우자와의 관계
- 자기 부모와의 관계
- 자녀와의 관계
- 배우자 부모와의 관계

**조사 기관:** 통계청
**조사 시점:** 2024년 5월 15일~30일
**표본:** 18,576가구 / 13세 이상 36,000명
**공표주기:** 2년

## AIPA 검증 방법

### 시뮬레이션 설정
- 페르소나 500명 생성 (KOSIS 연령/성별 분포 따름)
- 동일 질문/보기 입력
- 응답 분포 집계 → 통계청 결과와 비교

### 정량 지표
1. **카테고리별 절대오차**: |AIPA% - 실제%| 합 / 5
2. **KL Divergence**: 두 분포의 차이
3. **만족 합산 비교**: (매우+약간) 만족 비율

## 예상 발표 슬라이드 문구

> "통계청 사회조사 2024 실제 응답 분포(36,000명)와 AIPA 시뮬레이션 결과(500명)를 비교한 결과,
> 전 연령대 평균 KL Divergence 0.0X, 카테고리별 평균 오차 X.X%p로 실제 분포에 근접한 응답을 생성함을 확인."

연령대별 트렌드(연령↑ → 만족도↓) 재현 여부는 핵심 검증 포인트.
